O que é a tag Article?

Utilizamos o elemento

quando precisamos declarar um conteúdo que não precisa de outro para fazer sentido em um documento HTML, por exemplo, um artigo em um blog. É recomendado identificar cada 'Article' com um título.

O que é a tag Section?

O elemento HTML "section" representa uma seção genérica contida em um documento HTML, geralmente com um título, quando não existir um elemento semântico mais específico para representá-lo.
